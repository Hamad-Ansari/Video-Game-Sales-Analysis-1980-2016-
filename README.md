## 📊 Project Overview
An exploratory data analysis of video game sales data from 1980-2016, examining trends across platforms, genres, publishers, and regional sales patterns.

## 🔍 Key Insights
- Sales distribution across regions (NA, EU, JP, Other)
- Platform popularity trends
- Genre analysis
- Top publishers and games
- Temporal sales patterns

## 📈 Technologies Used
- **R 4.x**
- **tidyverse** (dplyr, ggplot2)
- **cowplot**, **ggthemes**
- **DescTools**
